Дана програма призначена для генерації PDF файлів для стікерів та шильдів на виробництві.
Суттєво зпрощує роботу оператора та знижує ризики помилок.
Зручний та простий дизайн робить процес приємним та швидким.

1. Запустити термінал "Командний рядок".
2. Перевірити чи встановлений Pyton.
3. Перевірити чи встановлений pip.
4. pip install openpyxl.
5. pip install fpdf2
6. pip install pywin32
7. pip install fpdf
8. pip install PyPDF2
9. Запустити stickerMaker.pyw
